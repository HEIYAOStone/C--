本文列出了最新版本的Microsoft Visual C --的下载链接。

Visual Studio 2015、2017和2019

下载适用于Visual Studio 2015、2017和2019的Microsoft Visual C --可再发行发行。以下更新是适用于Visual Studio 2015、2017和2019的最新支持的Visual C --可再发行发行。包括的是通用C运行时的基准版本，请参见MSDN了解详细信息。

  • x86：vc_redist.x86.exe

  • x64：vc_redist.x64.exe

  • ARM64：  vc_redist.arm64.exe

注意 Visual C -- 2015、2017和2019都共享相同的可再发行软件文件。

例如，安装Visual C -- 2019可再发行软件还将会影响使用Visual 2015和2017生成的程序。但是，安装Visual C -- 2015可再发行软件将不会取代Visual C-- 2017和2019可在发行软件所安装文件的较新版本。

这都是所有以前的Visual C --版本，因为它们都有自己独特的运行时文件，而不是与其他版本共享。

Visual C--可再发行程序包

Visual C --可再发行程序包在未安装中Visual C --的计算机上安装Visual C --库的运行时组件。运行使用相应版本的Visual C --开发的应用程序时需要这些库。

 

对于Visual Studio 2020

这些程序包安装C Runtime（CRT），标准C --，ATL，MFC，OpenMP和MSDIA库的运行时组件。它们安装在本机程序集缓存中，也称为WinSxS文件夹。的Windows操作系统版本上，适用于支持并行部署模型（CRT，STL，ATL，MFC，OpenMP）的库。
